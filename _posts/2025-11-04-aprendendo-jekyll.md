Hoje dei um passo importante na minha jornada como desenvolvedor: criei meu próprio blog usando Jekyll!

## O que é Jekyll?

Jekyll é um gerador de sites estáticos que transforma arquivos Markdown em um site HTML completo. É perfeito para blogs porque:

- É gratuito e open source
- Funciona perfeitamente com GitHub Pages
- Usa Markdown (fácil de escrever)
- É rápido e seguro (sem banco de dados)

## O que aprendi hoje

1. **Instalação do Jekyll no Debian**: Configurar Ruby e dependências
2. **Estrutura de arquivos**: Como organizar posts, páginas e configurações
3. **Git e GitHub**: Controle de versão e deploy automático
4. **Markdown**: Sintaxe simples para formatação de texto
